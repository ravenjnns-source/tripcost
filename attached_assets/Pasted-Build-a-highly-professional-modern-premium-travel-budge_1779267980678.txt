Build a highly professional, modern, premium travel budget calculator website called “TripCost”.

- The website name must be: TripCost

  - deep navy blue
  - bright teal / cyan
  - small green accent
- The whole UI should feel clean, elegant, premium, and very organized

- Make the visual presentation amazing, balanced, modern, and professional

Tech requirements:
- Use React
- Use Tailwind CSS
- No backend
- No database
- No paid APIs
- Keep the code clean and easy to maintain

Website purpose:
TripCost helps users estimate how much money they need to travel to a country based on:
- destination country
- number of travel days
- travel style:
  Budget / Standard / Luxury

Main calculator behavior:
- User selects a country
- User enters number of travel days
- User selects travel style
- On clicking Calculate, show:
  - Hotel total
  - Food total
  - Local transport total
  - Activities total
  - Visa cost
  - Flight estimate
  - Emergency extra
  - Grand total
  - Daily average cost
  - Smart verdict:
    Very affordable / Moderate / Expensive

Data structure:
- Create a separate file:
  src/data/travelCountries.js
- Import data from that file
- For now, use sample data structure that can later be expanded easily
